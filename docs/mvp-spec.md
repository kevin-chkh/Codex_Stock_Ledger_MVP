# 股票交易帳本 MVP Spec

## 產品目標

建立一個個人股票交易帳本網頁，用於記錄多個投資帳本、資金異動、買賣紀錄，並自動計算投資績效與持股分布。

第一版不追求完整券商級功能，而是先做到：

- 資料能正確記錄
- 成本與損益能正確計算
- 帳本狀態能快速理解
- 持股分布能視覺化查看
- 未來可擴充即時股價、股利、券商匯入等功能

## 使用者角色

單一個人使用者。

第一版暫不做多人帳號、權限管理、雲端同步。資料可以先存在本機瀏覽器或單機資料庫，後續再依技術方案決定。

## 核心物件

### 1. 帳本 Portfolio

欄位建議：

- 帳本 ID
- 帳本名稱
- 初始金額
- 目前現金餘額
- 累計投入金額
- 累計轉出金額
- 幣別
- 建立時間
- 更新時間
- 備註

帳本支援：

- 新增帳本
- 編輯帳本名稱、備註、幣別
- 資金加入
- 資金轉出
- 金額修正
- 查看資金異動紀錄

### 2. 資金異動 Cash Movement

類型：

- 加入金額
- 轉出金額
- 金額修正

欄位：

- 所屬帳本
- 異動類型
- 金額
- 異動後現金餘額
- 日期
- 備註

規則：

- 加入金額會增加現金餘額與累計投入金額
- 轉出金額會減少現金餘額與累計轉出金額
- 金額修正只調整現金餘額，不影響累計投入/轉出，並需留下備註

### 3. 股票 Stock

欄位：

- 股票代號
- 股票名稱
- 市場，例如 TW / US
- 產業分類
- 自訂標籤
- 目前價格
- 價格更新時間

第一版的目前價格採手動輸入。

### 4. 交易紀錄 Trade

欄位：

- 所屬帳本
- 股票代號
- 股票名稱
- 買入 / 賣出
- 交易日期
- 股數
- 成交單價
- 成交總額
- 手續費
- 交易稅
- 總成本 / 實收金額
- 備註

費用規則：

- 第一版預設使用台股規則
- 買入：成交金額 + 手續費
- 賣出：成交金額 - 手續費 - 交易稅
- 手續費預設：成交金額 x 0.1425%
- 交易稅預設：賣出成交金額 x 0.3%
- 費用可手動覆寫
- 可支援零股
- 暫不處理融資融券、當沖優惠、ETF 不同稅率

### 5. 持股 Position

由交易紀錄自動計算，不手動新增。

欄位：

- 所屬帳本
- 股票代號
- 股票名稱
- 持有股數
- 平均成本
- 累計買入成本
- 已回收金額
- 目前價格
- 目前市值
- 未實現損益
- 未實現報酬率
- 已實現損益
- 總損益
- 總報酬率
- 產業
- 標籤

成本計算：

- 第一版採平均成本法
- 買入後更新平均成本
- 賣出時依平均成本計算已實現損益
- 賣出股數不得超過目前持有股數

## 主要功能

顯示全部帳本或單一帳本總覽。

指標：

- 帳本總資產：現金 + 持股市值
- 現金餘額
- 持股市值
- 累計投入金額
- 累計轉出金額
- 已實現損益
- 未實現損益
- 總損益
- 總報酬率

圖表：

- 持股市值占比圓餅圖
- 產業分布圓餅圖
- 自訂標籤分布圓餅圖

列表：

- 持股排行
- 損益排行
- 最近交易紀錄

### B. 帳本管理

功能：

- 新增帳本
- 編輯帳本
- 切換帳本
- 加入金額
- 轉出金額
- 更新/修正金額
- 查看資金異動紀錄

驗證：

- 帳本名稱必填
- 金額需大於 0
- 轉出金額不可超過現金餘額，除非使用金額修正

### C. 交易管理

功能：

- 新增買入交易
- 新增賣出交易
- 查看交易列表
- 編輯交易
- 刪除交易
- 根據交易重新計算持股與損益

交易輸入：

- 帳本
- 股票代號
- 股票名稱
- 交易類型
- 日期
- 股數
- 成交單價
- 手續費
- 交易稅
- 備註

驗證：

- 股票代號或名稱至少填一個
- 股數大於 0
- 單價大於 0
- 賣出股數不可超過持有股數
- 買入時現金不足要提示，可先允許儲存但標記現金為負，這點可在技術方案前再決定

### D. 持股管理

功能：

- 查看目前持股
- 查看單一股票明細
- 手動更新目前價格
- 設定產業分類
- 設定自訂標籤
- 查看平均成本
- 查看已實現/未實現損益

持股列表欄位：

- 股票代號 / 名稱
- 持有股數
- 平均成本
- 目前價格
- 目前市值
- 未實現損益
- 未實現報酬率
- 已實現損益
- 總報酬率
- 產業
- 標籤

### E. 分析頁

第一版分析重點：

- 依股票看市值占比
- 依產業看市值占比
- 依標籤看市值占比
- 依股票看損益排行
- 依產業看損益排行

圖表形式：

- 圓餅圖
- 長條圖
- 表格

### F. 匯出與備份

第一版至少支援：

- 匯出交易紀錄 CSV
- 匯出資金異動 CSV
- 匯出目前持股 CSV

可選：

- 匯出整份 JSON 備份
- 從 JSON 還原

## 計算規則

### 買入

```text
成交金額 = 股數 x 成交單價
手續費 = 成交金額 x 手續費率
買入總成本 = 成交金額 + 手續費
現金餘額 -= 買入總成本
```

### 賣出

```text
成交金額 = 股數 x 成交單價
手續費 = 成交金額 x 手續費率
交易稅 = 成交金額 x 交易稅率
實收金額 = 成交金額 - 手續費 - 交易稅
賣出成本 = 賣出股數 x 平均成本
已實現損益 = 實收金額 - 賣出成本
現金餘額 += 實收金額
```

### 平均成本

```text
買入後平均成本 = 目前總持有成本 / 目前持有股數
賣出不改變剩餘持股平均成本
```

### 未實現損益

```text
目前市值 = 持有股數 x 目前價格
未實現損益 = 目前市值 - 剩餘持股成本
未實現報酬率 = 未實現損益 / 剩餘持股成本
```

### 總報酬率

```text
總損益 = 已實現損益 + 未實現損益
總報酬率 = 總損益 / 累計投入金額
```

總報酬率分母後續可再討論是否改成「目前投入本金」或「加權資金投入」。

## 第一版不包含

為了控制範圍，MVP 暫不包含：

- 即時股價 API
- 自動抓股票名稱與產業
- 股利、配息、配股
- 股票分割、減資
- 多使用者登入
- 雲端同步
- 券商對帳單匯入
- 美股匯率換算
- 融資融券
- 當沖與不同商品稅率
- 行動 App

## 設定項

第一版可以放：

- 預設手續費率
- 預設交易稅率
- 最低手續費
- 預設市場
- 預設幣別
- 是否允許現金餘額為負
- 匯出/匯入備份

## 成功標準

第一版完成後，使用者應該能做到：

1. 建立一個或多個帳本
2. 對帳本加入或轉出資金
3. 新增買入與賣出交易
4. 系統自動更新現金、持股、平均成本
5. 查看已實現與未實現損益
6. 手動更新股價後查看即時估算報酬率
7. 用產業與自訂標籤查看資產分布
8. 匯出交易與持股資料
